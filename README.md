# My Terminal Configs
## Intro
Some concise configs to my terminal tools.

## Note
- Install these tools before using.
- Clone this repo and rename it to `.config/`.
- Copy these to `.tmux.conf` in root directory.

```ini
# .tmux.conf
source-file ~/.config/tmux/tmux.conf
source-file ~/.config/tmux/statusline.conf
```
